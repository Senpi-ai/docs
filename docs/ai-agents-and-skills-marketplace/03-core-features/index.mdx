# Core Features

On top of the basic features offered by the Eliza Framework, [Moxie Eliza Skills Framework](https://github.com/moxie-protocol/moxie-agent-skills) repository also provides a set of core features that can be used to build more advanced and complex Skills.

![Moxie Eliza Skills Framework](/img/ai-agent-skills-architecture.png)

Those core features unique to the [Moxie Eliza Skills Framework](https://github.com/moxie-protocol/moxie-agent-skills) are:

- [**User Agent Wallets**](./01-user-agent-wallet.mdx): A user-owned embedded wallet that agents can use to execute onchain actions (e.g., trades, staking).
- [**Moxie Agent Lib**](./02-moxie-agent-lib.mdx): Utilities for executing transactions via the agent wallet & fetching Moxie data.
- [**Moxie Client**](./03-moxie-client.mdx): Injects user data into states, available globally.
