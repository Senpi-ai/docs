# Get All Auctions Participated By Certain User

In this tutorial, you will learn how to get all the auctions participated by a certain user by calling the Moxie Auction Subgraph API.

<InstallDependencies />

## Step 2: Construct The GraphQL Query

Then, you can get all the auctions participated by a certain user by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($userAddress: Bytes!) {
  users(where: { address: $userAddress }) {
    participatedAuctions {
      auctionId
      auctioningToken {
        id
        symbol
        decimals
      }
      txHash
    }
    address
  }
}
```

</TabItem>

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code and call the API with the `graphql-request` library:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery($userAddress: Bytes!) {
    users(where: { address: $userAddress }) {
      participatedAuctions {
        auctionId
        auctioningToken {
          id
          symbol
          decimals
        }
        txHash
      }
      address
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/auction/v3.4.3"
);

try {
  const data = await graphQLClient.request(query, {
    userAddress: "0x1a45675bd474c8e3056a7dbd19e738d7849c0d26",
  });
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
