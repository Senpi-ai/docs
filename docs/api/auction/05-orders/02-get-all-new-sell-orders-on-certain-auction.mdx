# Get All New Sell Orders On Certain Auction

In this tutorial, you will learn how to get all the new sell orders on a certain auction by calling the Moxie Auction Subgraph API.

## Step 2: Construct The GraphQL Query

Then, you can get all the new sell orders on a certain auction by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($auctionId: BigInt!) {
  newSellOrders(where: { auctionId: $auctionId }) {
    buyAmount
    id
    sellAmount
    userId
    blockInfo {
      blockNumber
      hash
      timestamp
    }
  }
}
```

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code and call the API with the `graphql-request` library:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery($auctionId: BigInt!) {
    newSellOrders(where: { auctionId: $auctionId }) {
      buyAmount
      id
      sellAmount
      userId
      blockInfo {
        blockNumber
        hash
        timestamp
      }
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/auction/v3.4.3"
);

try {
  const data = await graphQLClient.request(query, {
    auctionId: "1",
  });
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
