# Get Auction Summary

In this tutorial, you will learn how to get the Moxie protocol's auction summary data, which includes:

- total auctions
- total bidding value
- total orders
- etc.

by calling the Moxie Auction Subgraph API.

# Pre-requisites

## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get Moxie protocol's auction summary"
  subgraphLink={url.auction_subgraphs_legacy}
  query={`query MyQuery {
  summaries {
    totalAuctions
    totalBiddingValue
    totalOrders
  }
}`}
