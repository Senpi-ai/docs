# Get User's Portfolio

In this tutorial, you will learn how to get the user's Moxie portfolio by calling Moxie Protocol Subgraph API.

<PreRequisites />

<InstallDependencies />

Then, you can get the user's portfolio by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($userAddress: ID) {
  users(where: { id: $userAddress }) {
    portfolio {
      balance
      protocolTokenSpent
      subject {
        name
        symbol
      }
    }
  }
}
```

</TabItem>
