## Step 2: Construct The GraphQL Query

Then, you can get all the fan tokens owned by users by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($userAddress: ID, $fanTokenAddress: String) {
  users(where: { id: $userAddress }) {
    portfolio(where: { subject: $fanTokenAddress }) {
      balance
      protocolTokenSpent
      subject {
        name
        symbol
      }
    }
  }
}
```

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code and call the API with the `graphql-request` library:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery($userAddress: ID, $fanTokenAddress: String) {
    users(where: { id: $userAddress }) {
      portfolio(where: { subject: $fanTokenAddress }) {
        balance
        protocolTokenSpent
        subject {
          name
          symbol
        }
      }
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/protocol/v3.1.9"
);

try {
  const data = await graphQLClient.request(query, {
    userAddress: "0x20e04003c2edfbca86300f6b2e6736142718dae3",
    fanTokenAddress: "0xf07fbe409a57c4cf896ed24c5a438058a7abe39c",
  });
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
