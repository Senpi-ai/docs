# Get User's Sell Orders

In this tutorial, you will learn how to get the user's sell orders on the Moxie protocol by calling the Moxie Protocol Subgraph API.

## Step 2: Construct The GraphQL Query

Then, you can get the user's sell orders by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($userAddress: ID) {
  users(where: { id: $userAddress }) {
    sellOrders {
      price
      protocolToken
      protocolTokenAmount
      protocolFeeTransfer {
        txHash
        amount
      }
      subjectToken {
        name
        symbol
      }
      subjectAmount
      subjectFeeTransfer {
        txHash
        amount
      }
    }
  }
}
```

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery($userAddress: ID) {
    users(where: { id: $userAddress }) {
      sellOrders {
        price
        protocolToken
        protocolTokenAmount
        protocolFeeTransfer {
          txHash
          amount
        }
        subjectToken {
          name
          symbol
        }
        subjectAmount
        subjectFeeTransfer {
          txHash
          amount
        }
      }
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/protocol/v3.1.9"
);

try {
  const data = await graphQLClient.request(query, {
    userAddress: "0x20e04003c2edfbca86300f6b2e6736142718dae3",
  });
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
