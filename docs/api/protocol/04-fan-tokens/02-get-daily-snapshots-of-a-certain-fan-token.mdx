# Get Daily Snapshots Of Certain Fan Token

In this tutorial, you will learn how to get daily snapshots of a certain fan token by calling the Moxie Protocol Subgraph API.

<InstallDependencies />

## Step 2: Construct The GraphQL Query

Then, you can get the daily snapshots of a certain fan token by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($fanTokenAddress: ID) {
  subjects(where: { id: $fanTokenAddress }) {
    dailySnapshots {
      dailyBeneficiaryFeeChange
      dailyPriceChange
      dailyProtcolFeeChange
      dailyUniqueHoldersChange
      dailyVolumeChange
      endBeneficiaryFee
      endPrice
      endProtcolFee
      endTimestamp
      endUniqueHolders
      endVolume
      reserve
      startBeneficiaryFee
      startPrice
      startProtcolFee
      startUniqueHolders
      startVolume
      totalSupply
    }
  }
}
```

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery($fanTokenAddress: ID) {
    subjects(where: { id: $fanTokenAddress }) {
      dailySnapshots {
        dailyBeneficiaryFeeChange
        dailyPriceChange
        dailyProtcolFeeChange
        dailyUniqueHoldersChange
        dailyVolumeChange
        endBeneficiaryFee
        endPrice
        endProtcolFee
        endTimestamp
        endUniqueHolders
        endVolume
        reserve
        startBeneficiaryFee
        startPrice
        startProtcolFee
        startUniqueHolders
        startVolume
        totalSupply
      }
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/protocol/v3.1.9"
);

try {
  const data = await graphQLClient.request(query, {
    fanTokenAddress: "0x035eb3b4230b980723c71d3c09d6561824f1610b",
  });
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
