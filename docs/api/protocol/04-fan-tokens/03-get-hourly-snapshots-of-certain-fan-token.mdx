## Step 1: Add The API Query To Your Code

  subjects(where: { id: $fanTokenAddress }) {
    hourlySnapshots {
      hourlyBeneficiaryFeeChange
      hourlyPriceChange
      hourlyProtcolFeeChange
      hourlyUniqueHoldersChange
      hourlyVolumeChange
      endBeneficiaryFee

      endVolume
      reserve
      startBeneficiaryFee
      startPrice
      startProtcolFee
      startUniqueHolders
      startVolume
      totalSupply

  }
}`}
