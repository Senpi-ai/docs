## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get all the bonding curves initialized on the Moxie protocol within a certain time range"
  subgraphLink={SUBGRAPH_URL.protocol}
  query={`query MyQuery($startTime: BigInt, $endTime: BigInt) {
  moxieBondingCurveBondingCurveInitializedTxes(
    where: {
      blockInfo_: { timestamp_gte: $startTime, timestamp_lte: $endTime }
    }
  ) {
    initialSupply
    reserve
    reserveRatio
    subject
    subjectToken
    txHash
    blockInfo {
      blockNumber
      hash
      timestamp
    }
  }
}`}

  "endTime": "1720640000"
}`}
