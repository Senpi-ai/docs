## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get all the purchase transactions on the Moxie bonding curves within a certain time range"
  subgraphLink={SUBGRAPH_URL.protocol}
  query={`query MyQuery($startTime: BigInt, $endTime: BigInt) {
  moxieBondingCurveSubjectSharePurchasedTxes(
    where: {
      blockInfo_: { timestamp_gte: $startTime, timestamp_lte: $endTime }
    }
  ) {
    beneficiary
    buyAmount
    subject
    sellAmount
    sellToken
    txHash
    blockInfo {
      blockNumber
      hash
      timestamp
    }
  }
}`}
