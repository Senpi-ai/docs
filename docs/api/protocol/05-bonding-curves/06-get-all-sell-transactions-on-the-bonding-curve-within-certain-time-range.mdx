## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get all the sell transactions on the bonding curves within a certain time range"
  subgraphLink={SUBGRAPH_URL.protocol}
  query={`query MyQuery($startTime: BigInt!, $endTime: BigInt!) {
  moxieBondingCurveSubjectShareSoldTxes(
    where: {blockInfo_: {timestamp_gte: $startTime, timestamp_lte: $endTime}}
  ) {
    beneficiary
    buyAmount
    buyToken
    subject
    sellAmount
    sellToken
    txHash
    blockInfo {
      blockNumber
      hash
      timestamp
    }
  }
}`}
