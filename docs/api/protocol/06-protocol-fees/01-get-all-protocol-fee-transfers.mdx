# Get All Protocol Fee Transfers

In this tutorial, you will learn how to get all the protocol fee transfers on the Moxie protocol by calling the Moxie Protocol Subgraph API.

<InstallDependencies />

## Step 2: Construct The GraphQL Query

Then, you can get all the protocol fee transfers on the Moxie protocol by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery {
  protocolFeeTransfers {
    amount
    beneficiary {
      beneficiary
      totalFees
    }
    txHash
    order {
      orderType
    }
    subject {
      address: id
    }
  }
}
```

</TabItem>

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery {
    protocolFeeTransfers {
      amount
      beneficiary {
        beneficiary
        totalFees
      }
      txHash
      order {
        orderType
      }
      subject {
        address: id
      }
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/protocol/v3.1.9"
);

try {
  const data = await graphQLClient.request(query);
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
