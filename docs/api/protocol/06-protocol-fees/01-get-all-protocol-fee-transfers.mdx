## Step 1: Add The API Query To Your Code

  useCase="get all the protocol fee transfers on the Moxie protocol"
  subgraphLink={SUBGRAPH_URL.protocol}
  query={`query MyQuery {
  protocolFeeTransfers {
    amount
    beneficiary {
      beneficiary
      totalFees
    }
    txHash
    order {
      orderType
    }
    subjectToken {
      address: id
