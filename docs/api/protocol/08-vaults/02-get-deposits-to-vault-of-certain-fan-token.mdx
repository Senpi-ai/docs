# Get Deposits To Vault Of Certain Fan Token

In this tutorial, you will learn how to get all the Moxie deposits to the vault contract by a certain fan token on the Moxie protocol by calling the Moxie Protocol Subgraph API.

## Step 2: Construct The GraphQL Query

Then, you can get all the Moxie deposits to the vault contract by a certain fan token on the Moxie protocol by calling the subgraph API with the following query:

<Tabs>
<TabItem value="query" label="Query">

```graphql
query MyQuery($fanTokenAddress: Bytes) {
  vaultDepositTxes(where: { subject: $fanTokenAddress }) {
    amount
    sender
    subject
    token
    totalReserve
    txHash
    blockInfo {
      blockNumber
      hash
      timestamp
    }
  }
}
```

## Step 3: Add The API Query To Your Code

With this GraphQL query, you can add it to your source code:

<Tabs>
<TabItem value="grapqhl_ts" label="graphql-request (TS)">

```tsx
import { gql, GraphQLClient } from "graphql-request";

const query = gql`
  query MyQuery($fanTokenAddress: Bytes) {
    vaultDepositTxes(where: { subject: $fanTokenAddress }) {
      amount
      sender
      subject
      token
      totalReserve
      txHash
      blockInfo {
        blockNumber
        hash
        timestamp
      }
    }
  }
`;

const graphQLClient = new GraphQLClient(
  "https://api.studio.thegraph.com/proxy/27864/protocol/v3.1.9"
);

try {
  const data = await graphQLClient.request(query, {
    fanTokenAddress: "0xf07fbe409a57c4cf896ed24c5a438058a7abe39c",
  });
  console.log(data);
} catch (e) {
  throw new Error(e);
}
```

</TabItem>
