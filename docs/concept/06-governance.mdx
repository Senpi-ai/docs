# Governance

**MOXIE Holders are able to participate in governance of the Moxie Protocol**

**Voting rights are provided as follows:**

| Token Bucket                                 | Voting Power/Token | Notes                                                                    |
| -------------------------------------------- | ------------------ | ------------------------------------------------------------------------ |
| Vesting Tokens                               | 1                  |                                                                          |
| Circulating Tokens                           | 1                  |                                                                          |
| Tokens locked in the protocol Bonding Curves | 2                  | Tokens locked in the protocol Bonding Curve carry 2 points voting power. |

All MOXIE holders can propose and vote on governance proposals to change a subset of network parameters.

The following parameters are currently changeable by governance votes:

| Parameter                                                                                                                | Notes                         |
| ------------------------------------------------------------------------------------------------------------------------ | ----------------------------- |
| Use of Protocol Fees from Protocol Transactions                                                                          | Currently set to burn at 100% |
| Inflation adjustments in Rewards relative to growth of the Farcaster Network or increases/decreases in MOXIE token value |                               |
| Grant programs                                                                                                           |                               |
| Common Goods Rewards Grants                                                                                              |                               |
| Creating new types of Fan Tokens                                                                                         |                               |
| Upgrading the Protocol to a new version                                                                                  |                               |
