# Dashboards

At launch a selection of Moxie Protocol Economy dashboards can be found on Airstack, including:

- Upcoming Fan Token Auctions

- TLV in the Moxie Protocol, aggregated totals and for each Fan Token

- Prices for each Fan Token

- Holders of each Fan Token

- User-specific Fan Token portfolio details (requires login for each user)

Anyone is free to create their own dashboards using the [Moxie Protocol Subgraphs](https://build.moxie.xyz/the-moxie-protocol/subgraphs) and datasets.
