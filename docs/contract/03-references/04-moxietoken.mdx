# MoxieToken.sol

The `MoxieToken` is an ERC20 contract that inherits OpenZeppelin's `ERC20`, `ERC20Burnable`, and `ERC20Permit` libraries.

It has the following token metadata:

- Name: Moxie
- Symbol: MOXIE
- Decimals: 18
- Total Supply: 10,000,000,000
