# Overview

Moxie offers a curated collection of publicly available Dune tables designed to provide seamless access to comprehensive data on [Based Rewards](../learn/technical-deep-dive/04-based-rewards.mdx).

These tables are tailored to meet the needs of both users and developers, enabling them to effortlessly explore, access, and analyze both current and historical [Based Rewards](../learn/technical-deep-dive/04-based-rewards.mdx) data.

Below is a comprehensive list of tables available for your use:

| Table                                                     | Description                                                                                                                                                                                                                                                                  |
| --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [User Rewards Live](https://dune.com/queries/4510714)     | This table provides estimated [Based Rewards](../learn/technical-deep-dive/04-based-rewards.mdx) that Moxie users will earn based on their gas usage on that day. <br /><br /> This table is sorted by rewards earned in descending order, making it ideal as a leaderboard. |
| [User Daily Live](https://dune.com/queries/4516789)       | This table provides the amount of [Based Rewards](../learn/technical-deep-dive/04-based-rewards.mdx) earned by Moxie users the previous day. Rewards will be calculated daily at 00:00 UTC.                                                                                  |
| [User Snapshot](https://dune.com/queries/4468458)         | This table provides historical snapshot data of [Based Rewards](../learn/technical-deep-dive/04-based-rewards.mdx) earned by all Moxie users since Moxie expansion on Base.                                                                                                  |
| [Allowlisted Contracts](https://dune.com/queries/4516518) | This table provides the list of all allowlisted contracts for Based Rewards with their corresponding multipliers. For every transactions user made through these allowlisted contracts, they'll earn multipliers on their gas calculations.                                  |

These tables are thoughtfully organized and readily accessible, allowing you to dive into the data effortlessly. Whether you are analyzing trends, extracting insights, or building applications, these resources are here to support your needs.
