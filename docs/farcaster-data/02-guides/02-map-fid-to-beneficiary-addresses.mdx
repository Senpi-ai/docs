# Map FID to Beneficiary Addresses

Using data from the [`moxie_resolve.json`](/farcaster-data/guides/download-data) file, you can easily import it and map an FID to a beneficiary address very easily using the function defined below:

<Tabs>
  <TabItem value="ts" label="TypeScript">

```ts title="index.ts"
import data from "./moxie_resolve.json";

const resolveFidToBeneficiary = (fid: number) => {
  return data
    ?.filter((d) => d?.fid === fid && d?.type === "WALLET_ADDRESS")
    ?.map((d) => d?.address);
};
```

  </TabItem>
  <TabItem value="js" label="JavaScript">

```js title="index.js"
const data = require("./moxie_resolve.json");

const resolveFidToBeneficiary = (fid) => {
  return data
    ?.filter((d) => d?.fid === fid && d?.type === "WALLET_ADDRESS")
    ?.map((d) => d?.address);
};
```
