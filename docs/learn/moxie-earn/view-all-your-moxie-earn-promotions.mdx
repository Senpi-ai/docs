# View All Your Moxie Earn Promotions

To view all the Moxie Earn promotions you have created, you can simply click the `Your Promotions` button on the Moxie Earn mini-app. You can access the Moxie Earn mini-app from the Moxie Cast Action or from the Moxie Earn Composer Action when writing a cast.

</div>
