# `moxieWalletClient`

## Description

`moxieWalletClient` state returns a class instance of Senpi agent's embedded wallet that can be used to execute transactions and sign messages.

## Usage

```ts
import { MoxieWalletClient } from "@moxie-protocol/moxie-agent-lib";
const moxieWalletClient: MoxieWalletClient = state.moxieWalletClient;
```
