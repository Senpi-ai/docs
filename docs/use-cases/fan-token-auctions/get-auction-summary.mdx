# Get Auction Summary

In this tutorial, you will learn how to get the Moxie protocol's auction summary data, which includes:

- total auctions
- total bidding value
- total orders
- etc.

by calling the Moxie Auction Subgraph API.

# Pre-requisites

<Prerequisites apiName="Moxie Auction Subgraph" />

## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get Moxie protocol's auction summary"
  subgraphLink={url.auction_stats_subgraphs}
  query={`query MyQuery {
  summaries {
    totalAuctions
    totalBiddingValue
    totalOrders
  }
}`}
  response={`{
  "data": {
    "summaries": [
      {
        "totalAuctions": "41",
        "totalBiddingValue": "543157600000000000000000",
        "totalOrders": "65"

    ]
  }
}`}
/>
