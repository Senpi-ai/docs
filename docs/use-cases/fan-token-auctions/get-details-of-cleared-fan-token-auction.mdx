## Step 1: Add The API Query To Your Code

  useCase="get details of a cleared auction"
  subgraphLink={url.auction_stats_subgraphs}
  query={`query MyQuery($auctionId: BigInt!) {
  auctionCleareds(where: { auctionId: $auctionId }) {
    clearingPriceOrder
    soldAuctioningTokens
    soldBiddingTokens

}`}
