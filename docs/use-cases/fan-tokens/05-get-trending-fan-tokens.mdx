## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get all-time trending fan tokens sorted by the number of fan token buy volume"
  subgraphLink={url.protocol_stats_subgraphs}
  query={`query MyQuery {
  subjectTokens(orderBy: buySideVolume, orderDirection: desc) {
    buySideVolume
    currentPriceInMoxie
    currentPriceInWeiInMoxie
    address: id
    name
    symbol
    decimals
    createdAtBlockInfo {
      id
    }
  }
}`}
