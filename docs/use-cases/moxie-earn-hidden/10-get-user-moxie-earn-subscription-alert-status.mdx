# Get User's Moxie Earn Subscription Alert Status

<Header
  useCase="get user's Moxie Earn subscription alert status"
  apiName="Moxie Partner"
/>

When a user subscribes, they will receive Direct Cast alerts whenever they are eligible for new Moxie Earn promotions.

# Pre-requisites

1. Install dependencies `graphql`and`graphql-request` to your project :

<Install dependencies="graphql graphql-request" />

2. Your will need a special API key to access to the Moxie Partner API. To request access, please fill in this form [here](https://forms.gle/eRQ6uiCzica2J6qo8).

## Step 1: Add The API Query To Your Code

<AddQuery
  useCase="get user's Moxie Earn subscription alert status"
  subgraphLink="https://api.moxie.xyz/protocol/graphql"
  query={`query($fid: String!) {
  FetchSubscriptionInfo(input: { fid: $fid}) {
    promotionSubscription {
      createdAt
      fid
      id
      status
      updatedAt
    }
  }
}`}
  headers={`{
  // Add Airstack API key here
  "x-airstack-protocol": process.env.AIRSTACK_API_KEY as string,
}`}
  variable={`{
  "fid": "602",
}`}
  response={`{
  "data": {
    "FetchSubscriptionInfo": {
      "promotionSubscription": [
        {
          "createdAt": "2024-11-05 08:06:57.000",
          "fid": "602",
          "id": "a1bb3d5a-3757-4be3-baf5-c3755ef99542",
          "status": "ACTIVE",
          "updatedAt": "2024-11-05 08:06:57.000"
        }
      ]
    }
  }
}`}
/>
